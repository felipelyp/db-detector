# Db Detector
Um capturador de sinal de onu´s
> O aplicativo trabalha enviando comandos via telnet
### Disponível para:
- OLT FIberhome

### Como funciona?
O aplicativo se conecta através do protocolo Telnet
- Envia comandos
- Captura os resultados
- Verifica as resposta
